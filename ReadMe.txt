Request from the client should be of the form - prime startNumber endNumber

eg.  prime 2000 75000

Any other type of request is not understood by the server.

Other types of messages that are present and understood by the server-

1 -		.q == Terminate the client
2 -		message that start with .r  == It denotes that result of operation is being returned.


NOTE - Close the server program using Ctrl+C
